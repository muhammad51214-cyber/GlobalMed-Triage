# GlobalMedTriage Frontend

## Setup

1. Copy `.env.example` to `.env` and set backend URL if needed.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start dev server:
   ```bash
   npm run dev
   ```

## Features
- Voice-first UI (WebRTC)
- Real-time agent responses via WebSocket
- Modern React + TypeScript + Tailwind
